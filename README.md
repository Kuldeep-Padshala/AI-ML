## 📖 About This Repository

This repository is my personal AI/ML knowledge base and learning journey.

Instead of creating scattered notebooks and random folders, I wanted a single organized place where I can:

- Learn concepts systematically
- Store notes and study materials
- Implement algorithms from scratch
- Practice with datasets
- Build projects
- Experiment with new ideas
- Track my progress over time
- Share my learnings with the community

The goal is not just to learn how to use Machine Learning libraries, but to understand **how things work behind the scenes**.
