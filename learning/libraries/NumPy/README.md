# NumPy

NumPy is the foundation of the modern Python data science ecosystem.

Almost every Machine Learning and Deep Learning library—including Pandas, SciPy, Scikit-Learn, TensorFlow, and PyTorch—either builds upon or interacts with NumPy arrays.

This section focuses on understanding NumPy from an AI/ML perspective rather than simply learning its syntax.

The notebooks emphasize practical operations used daily in Machine Learning pipelines, including vectorized computation, indexing, reshaping, broadcasting, linear algebra, and random number generation.

By the end of this module, you should be comfortable manipulating numerical data efficiently and understand why NumPy is significantly faster than traditional Python lists.

> Mastering NumPy makes every subsequent AI/ML library easier to understand.
